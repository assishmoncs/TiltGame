TiltGame is a tilt-controlled arcade Android game built using Kotlin.

## Features
- Accelerometer-based movement
- Green orbs (+1 score)
- Red bonus orbs (+5 score, limited time)
- Background music and sound effects
- Vibration feedback
- High score saving using SharedPreferences
- Start, Help, About, and Settings screens
- Portrait mode optimized

## Tech Stack
- Kotlin
- Custom View rendering
- SensorManager (Accelerometer)
- MediaPlayer & SoundPool
- SharedPreferences
- Material Components

## How to Run
1. Clone the repository
2. Open in Android Studio
3. Build and run on emulator or physical device
